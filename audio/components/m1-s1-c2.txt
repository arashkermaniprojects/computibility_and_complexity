A word about encodings, because this trips students up. When we write angle-brackets around an object — angle-B for a DFA, angle-G, w for a CFG and a string — we mean some fixed reasonable encoding of the object as a string of zeros and ones. Sipser uses this notation throughout the book. Two things to know. First, the choice of encoding does not matter, as long as it is reasonable: any two reasonable encodings are convertible to each other in polynomial time. Second, the Turing machine that decides the language reads the encoded input from its tape and works with it like a program reads structured data. The encoding is just the bridge between mathematical objects and tape strings.
