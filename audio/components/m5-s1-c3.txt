Two equivalent definitions of N P. The first uses the verifier viewpoint: a language L is in N P when there exists a polynomial-time machine V such that x belongs to L exactly when some short certificate c convinces V to accept the pair x, c. The certificate must be polynomial in the input length so that the verifier even has time to read it. The second definition uses nondeterminism: N P equals the union of N TIME of n to the k, over all constants k. A nondeterministic Turing machine that runs in polynomial time decides exactly the languages in N P. These two phrasings are interchangeable. Given a verifier, an N T M can guess the certificate one symbol at a time and then run V. Going the other way, a polynomial-time accepting computation path of an N T M is itself the certificate. Sipser proves this equivalence in lecture fourteen and uses both viewpoints throughout the rest of the book.
