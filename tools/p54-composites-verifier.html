COMPOSITES is in NP · certificate is a divisor

Show a non-trivial divisor; verifier multiplies and checks.

CLO-3R2·30 ptsL10
1 · Theory

COMPOSITES = {n : n is composite}. Certificate for n: a non-trivial divisor d (1 < d < n). Verifier: confirm n mod d = 0. Polynomial in log n.

Why both COMPOSITES and PRIMES are in P

COMPOSITES ∈ NP (above). PRIMES ∈ NP via Pratt certificates. Both shown ∈ P by AKS (2002). For decades only NP membership was known.

3 · Q&A defence notes
  • Q1. Why is the certificate "small"?
    A divisor d satisfies d ≤ √n, so it has at most (log n)/2 bits. Polynomial in input size (which is log n).
  • Q2. Is the verifier polynomial?
    Yes: dividing n by d uses school long division in O((log n)²) bit operations.
  • Q3. How does AKS achieve P?
    A clever polynomial identity test mod p^k — not a divisor at all. Different witness structure.
