In this figure, we visualize a game tree for the TQBF problem, which stands for "True Quantified Boolean Formula." The game tree represents a formula like "there exists an x, for all y, there exists a z, such that φ(x, y, z) holds." As we evaluate this tree from the bottom up, we encounter existential nodes, represented as OR, and universal nodes, represented as AND. The key point is that while the tree can grow exponentially in size, it has polynomial depth. This allows us to use depth-first evaluation, which only requires polynomial space. Essentially, a TQBF formula is true if the player controlling the existential variables has a winning strategy. The takeaway? Efficient space usage in evaluating complex logical formulas is crucial in computational complexity.
