The Immerman–Szelepcsényi theorem is a fascinating result in computational complexity that tells us something surprising about nondeterministic log space, or NL. It shows that NL is equal to coNL, meaning that for every problem in NL, there's a way to verify that the answer is "no" using the same amount of space. A key example here is the UNREACHABILITY problem, which asks if there’s no path from a starting vertex to a target vertex in a directed graph.

To understand how this works, we can use an inductive counting technique. Imagine we define c sub i as the number of vertices reachable from a starting point s in i steps. We can compute these values step by step, starting from c sub 0 and going up to c sub n, where n is the total number of vertices. The clever part is the recurrence: for each new step, we check if a vertex is either already in the reach-set or has a predecessor in it. 

Now, to determine if a target vertex t is unreachable, we nondeterministically guess all reachable vertices and check each one. If t isn’t among them, then it must be unreachable. The beauty of this approach is that all operations stay within log space, making everything efficient. This theorem not only gives us insights into the structure of complexity classes but also reinforces the power of nondeterminism in log space.
