Here is the CFG G sub M, w used in the ALL-CFG reduction. The big universe rectangle at the top is the set of all strings over the alphabet. Inside it sits a small oval — the set of valid accepting computation histories of M on input w. G sub M, w generates everything OUTSIDE that oval and nothing inside it.

How? With three nondeterministic branches. The start symbol S nondeterministically picks one of three "ways to be a bad history" and generates a string with exactly that flaw. Branch one matches strings whose first configuration C-one is not q-zero followed by w — a CFG can check this because the start is finite and the rest is arbitrary. Branch three matches strings whose last configuration contains no accept state — again, a finite local check.

The trick is branch two, the bad-transition branch. To verify that two consecutive configurations don't follow by delta, the CFG writes every other configuration in reverse, so a context-free derivation can pair them cell-by-cell using a stack-like rule. This palindrome trick is what makes the construction work; without it, matching long configurations would need unbounded memory.

The conclusion: L of G sub M, w equals sigma star if and only if every string is bad, if and only if no valid accepting history exists, if and only if M does not accept w. Therefore deciding ALL-CFG would let us decide co-A-T-M. ALL-CFG is undecidable. This is Sipser theorem 5.13.
