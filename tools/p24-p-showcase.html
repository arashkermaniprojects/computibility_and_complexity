3 · Q&A defence notes
  • Q1. Why is PATH in P but HAMPATH probably not?
    PATH = "does ANY path exist" = simple reachability = BFS. HAMPATH = "does a path through ALL vertices exist" = combinatorial search.
  • Q2. Why is Euclid's gcd algorithm fast?
    Each step at least halves the smaller number (proven). So O(log min(a,b)) iterations — polynomial in bit length.
  • Q3. Why O(n³) for CYK?
    O(n²) table cells × O(n) split points each. Constant grammar.
