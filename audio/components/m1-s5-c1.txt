A sub CFG asks whether a context-free grammar G generates the string w. The naïve approach — try every derivation — does not halt, because derivations can grow arbitrarily. The fix is the CNF trick. Convert G to Chomsky Normal Form, where every rule is either A goes to BC or A goes to a single terminal. In CNF, a derivation of a string of length n uses exactly two n minus one productions — no more, no fewer. So the search space is finite: just try every derivation tree of that exact size. That bound is the key, and it is what makes the decider halt. From here you can either enumerate trees, or use the CYK dynamic-programming algorithm, which does the same search in cubic time.
