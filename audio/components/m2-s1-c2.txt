The diagonal lemma is the abstract engine behind everything in this module. State it like this: if you list any countable family of functions f one, f two, and so on, from the naturals to the set zero, one, then you can construct a new function g defined by g of n equals one minus f sub n of n. This g disagrees with each f sub i at the input i, so g is not anywhere in your list. The corollary is that the set of all such functions is uncountable. Diagonalization will reappear in slide three, where we use it not on functions but on Turing machines, to prove A sub TM is undecidable.
