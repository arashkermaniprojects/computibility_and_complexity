For E sub DFA — is the language of B empty? — we ask a graph question. A DFA's language is empty exactly when no accept state is reachable from the start state. So the algorithm is graph reachability. Mark the start state. Repeatedly: mark any state reachable in one transition from a state already marked. Stop when no new marks happen. If any marked state is accepting, the language is non-empty — reject. If none are, accept the emptiness claim. The procedure halts because there are only finitely many states to mark; it is correct because every reachable state gets marked. Standard breadth-first search.
