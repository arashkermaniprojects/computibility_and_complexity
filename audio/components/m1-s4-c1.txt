EQ sub DFA asks whether two DFAs A and B accept the same language. The algorithm uses a beautiful Sipser trick: reduce equivalence to emptiness. Build the symmetric-difference DFA C, whose language is the set of strings accepted by exactly one of A and B. If A and B agree everywhere, C accepts nothing. If they disagree on any string, C accepts at least one string. So L of A equals L of B if and only if L of C is empty. We already know how to decide emptiness from the previous slide. Construct C using the closure of regular languages under intersection, complement, and union. Then run the E sub DFA decider on C. Done.
