SAT search ≤p SAT decision · self-reducibility

A SAT decider lets you actually find a satisfying assignment in poly time.

CLO-3CLO-4R2·30 ptsL12
1 · Theory
Search-to-decision

Given a SAT decider D, find a satisfying assignment with n+1 calls to D:

while variables remain:
  pick variable x; set x = TRUE in φ
  if D(φ) accepts: x is TRUE; recurse on φ
  else: x = FALSE; recurse on φ|_{x=F}

So if P = NP, finding is as easy as deciding. The "search" question is no harder than the "decision" question.

3 · Q&A defence notes
  • Q1. Why does the decider on φ|x=T tell us x's value?
    If D says φ|x=T is SAT, there's a satisfying assignment with x=T; set x=T. Otherwise the original must have a satisfying assignment with x=F.
  • Q2. How many calls to D?
    n + 1 (one initial check, then n per-variable fixings). Polynomial in the original formula size.
  • Q3. Does this work for other NP-complete problems?
    Yes — SAT's self-reducibility is the canonical pattern; analogous tricks work for HAMPATH, CLIQUE, etc.
