TQBF · evaluate quantified formulas as a game tree

∃-nodes take OR, ∀-nodes take AND. Post-order evaluation.

CLO-4R3·40 ptsL13
1 · Theory

TQBF = {⟨φ⟩ : the quantified Boolean formula φ is true}. PSPACE-complete. Recursive evaluation = game between ∃ and ∀.

Evaluation

∃x. ψ(x) is true ⇔ ψ(0) ∨ ψ(1). ∀x. ψ(x) is true ⇔ ψ(0) ∧ ψ(1). Recurse until no quantifiers left, then evaluate the propositional kernel.

3 · Q&A defence notes
  • Q1. Why is TQBF PSPACE-complete and not just in NP?
    A SAT formula has only one quantifier prefix (existential). TQBF allows alternating ∀∃∀∃… — exponentially more complex.
  • Q2. How does the recursion use polynomial space?
    Depth-first evaluation reuses memory at each level. Only the current path's partial assignment is stored. Depth = n levels × O(1) bits each = O(n).
  • Q3. Total tree size?
    2n leaves. Walking them takes 2n time. PSPACE-completeness proves no faster algorithm is known.
