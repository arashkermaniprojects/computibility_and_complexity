1 · Theory
Big-O

f(n) = O(g(n)) iff ∃ c > 0, n₀ such that for all n ≥ n₀, f(n) ≤ c · g(n). Constants and lower-order terms are absorbed.

So 3n² + 5n + 7 = O(n²). The dominant term swallows the rest as n grows.

3 · Q&A defence notes
  • Q1. Why drop the constants?
    Big-O is about asymptotic behaviour. 3n² and n² differ by a factor that doesn't depend on n. The shape of growth is what we care about.
  • Q2. Big-O vs Theta?
    O is an upper bound; Θ is tight (both upper and lower). 3n² = O(n³) is true; 3n² = Θ(n³) is false.
  • Q3. What's "dominated by"?
    For n ≥ some n₀, the smaller-power terms become arbitrarily small relative to the leading term. So removing them changes the value by at most a fixed multiplicative factor.
