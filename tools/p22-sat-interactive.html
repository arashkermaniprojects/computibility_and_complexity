1 · Theory

SAT = {⟨φ⟩ : Boolean formula φ is satisfiable}. A clause is green (satisfied) iff at least one literal is true under the current assignment.

NP definition for SAT

Certificate = satisfying assignment; verifier checks each clause in O(m) time.

3 · Q&A defence notes
  • Q1. What's the worst case?
    2n trials. For n=20, that's a million; for n=30, a billion.
  • Q2. Is there a smart algorithm?
    DPLL, CDCL — practically fast but worst-case still exponential.
  • Q3. How does a verifier differ from a solver?
    Verifier gets the assignment; just checks. Solver searches.
