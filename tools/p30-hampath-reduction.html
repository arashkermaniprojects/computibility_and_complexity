3SAT → HAMPATH · gadget reduction

Variable diamonds + clause spreaders. Path = assignment.

CLO-4R2·30 ptsL12
1 · Theory

For each variable, build a "diamond" gadget that the Hamiltonian path must traverse left-to-right (= true) or right-to-left (= false). For each clause, a spreader node forces the path to visit one of its literals' diamonds.

Result

HAMPATH exists ⇔ 3SAT formula is satisfiable. Reduction is polynomial. HAMPATH is NP-complete.

3 · Q&A defence notes
  • Q1. What does "traversal direction = assignment" mean?
    Each variable diamond can be traversed left→right (variable = true) or right→left (variable = false). The whole path picks one direction per diamond.
  • Q2. How does the clause spreader work?
    For each clause C, a spreader node connects to the diamonds of variables appearing in C. The path must dip into the spreader and exit via one literal-true diamond — equivalent to "at least one literal satisfied".
  • Q3. Size of the gadget?
    O((n+m)²) nodes, O((n+m)²) edges. Polynomial.
