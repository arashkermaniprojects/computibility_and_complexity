Let’s walk through the proof of the space hierarchy theorem, focusing on two key claims we need to verify. First, we need to show that D doesn't belong to SPACE(f). So, imagine we assume the opposite: that D is in SPACE(f). This means D can be simulated by some machine M_k operating with f space. Now, if we run D on its own description, which is ⟨M_k⟩, M_k would produce some bit b. However, due to the construction of D, it’s designed to output the opposite bit, which would be 1 minus b. If D equals M_k, then we have a situation where b must equal 1 minus b. This leads us to a contradiction since the only bit satisfying that equation is 0. 

Now, onto Claim 2, which states that D is in SPACE(f log f). Here’s how we can simulate D: M uses f(n) tape cells for its computation. For the head position, state, and counter, we only need O(log f(n)) space. When we combine these, we get a total of O(f log f) space. This completes our proof, showing that more space indeed allows us to recognize strictly more languages.
