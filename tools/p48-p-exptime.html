P ⊊ EXPTIME · proved strict

Time hierarchy theorem: there is a problem in EXPTIME outside P.

CLO-5R3·40 ptsL16
1 · Theory
Theorem

By the time hierarchy theorem, for any constant k there exists a language in TIME(2n) \ TIME(nk). So P ⊊ EXPTIME.

This is one of the few PROVEN strict containments between standard complexity classes. P ⊊ NP, NP ⊊ PSPACE, PSPACE ⊊ EXPTIME — all open.

3 · Q&A defence notes
  • Q1. What's an explicit language in EXPTIME \ P?
    The "halt within 2ⁿ steps" diagonalizer from the time hierarchy proof. Concrete natural examples: generalized chess (EXPTIME-complete).
  • Q2. Why is P ⊊ EXPTIME provable but P ⊊ NP not?
    Hierarchy theorem compares time bounds within the same kind of machine. NP uses nondeterminism — a different machine model. We don't have a direct hierarchy theorem connecting deterministic and nondeterministic time.
  • Q3. Are there separations between P and PSPACE?
    Not known. P ⊊ PSPACE is open. If we could prove P ⊊ PSPACE we'd also resolve P vs NP (since NP ⊆ PSPACE).
