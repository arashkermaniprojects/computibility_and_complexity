TQBF₃ normal form

Every QBF is equivalent to one of the form ∃x₁ ∀x₂ ∃x₃ ∀x₄ … Q xn. φ(x₁,…,xn) where φ is 3-CNF and quantifiers strictly alternate.

Useful because: (1) standardised form for reductions; (2) PSPACE-completeness proofs target this restricted form.

3 · Q&A defence notes
  • Q1. How to enforce alternation?
    Add dummy variables: ∃x ∃y. ψ becomes ∃x ∀dummy ∃y. ψ. The dummy variable is forced to one specific value by adding it conjunctively, so the formula is unchanged but quantifiers alternate.
  • Q2. How to convert matrix to 3-CNF?
    Apply the same SAT → 3SAT trick (Problem 26) to the propositional kernel.
  • Q3. Total size?
    Polynomial. Adding alternation doubles variables; 3-CNF conversion linear blowup.
