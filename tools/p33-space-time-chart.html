1 · Theory
Inclusion theorem

For any reasonable f(n) ≥ log n: TIME(f) ⊆ SPACE(f) ⊆ TIME(2O(f)). First is trivial (can't visit more cells than steps). Second is because a SPACE(f) machine has at most 2O(f) configurations.

3 · Q&A defence notes
  • Q1. Why isn't TIME = SPACE?
    Space can be reused; time can't. A SPACE(f) algorithm can revisit the same cells exponentially many times, taking 2O(f) time.
  • Q2. Real examples of "space-cheap, time-expensive"?
    Branch-and-bound search trees: O(n) call stack, exponential running time.
  • Q3. Is the second inclusion tight?
    In general yes — Savitch's theorem TQBF runs in PSPACE but takes 2O(n) time.
