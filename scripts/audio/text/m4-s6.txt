Practice for Module 4 comes from Sipser sections seven point one and seven point two. Exercises to do: seven point five, which exercises Big-O classification; seven point thirteen, which constructs a polynomial-time algorithm for a graph problem; and seven point twenty, which is a meta-question about the closure properties of P. As you work through them, build an internal library of "P-style" algorithms — graph search, dynamic programming, divide and conquer, table look-up. When you encounter a new problem, your first move should be to check whether one of these techniques can dispatch it in polynomial time.
