For E sub CFG — is the language of grammar G empty? — we use a marking algorithm. Mark every terminal symbol. Then, repeatedly: mark a variable A if there is some rule A goes to alpha where every symbol in alpha is already marked. Stop when one full pass adds no new marks. If the start variable S got marked, then S derives some terminal string, so the language is non-empty. If S did not get marked, no terminal string is derivable from S, so the language is empty. The algorithm always halts — there are only finitely many variables, and each gets marked at most once. The procedure is the grammar analog of graph reachability from slide three.
