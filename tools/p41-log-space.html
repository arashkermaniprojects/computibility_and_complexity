Two pointers into the input, that's all the memory needed.

1 · Theory

L = class of problems decidable by a deterministic TM using O(log n) work-tape space. The input is on a read-only tape (doesn't count).

Palindrome ∈ L

Two pointers i (left) and j (right). Compare input[i] with input[j]. Advance toward middle. Each pointer takes ⌈log₂ n⌉ bits. Total: 2 log n + O(1) = O(log n).

3 · Q&A defence notes
  • Q1. Why is the input tape "free"?
    L is defined with two tapes: read-only input + work tape. Only the work tape counts toward the space bound. This matches the intuition of "pointers into the input."
  • Q2. Why 2 pointers fit in log n bits?
    Each pointer is an integer in [0, n-1], so ⌈log₂ n⌉ bits. Total for two pointers: 2 log n.
  • Q3. What's the time complexity?
    O(n) — each pair compared once. Polynomial in input size.
